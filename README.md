# NetSpy - Simple Packet Sniffer with Scapy

This is part of a 10-day project to learn Scapy and Wireshark.

## Day 1 - Basic Packet Sniffer

- Captures 10 packets from the network
- Displays summary of each packet
- Uses `scapy.sniff()`

## Requirements
- Python 3.8+
- Scapy
- Wireshark (optional)
